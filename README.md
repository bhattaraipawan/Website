# Personal Website
A clean, responsive portfolio website built with HTML and CSS to showcase my professional profile, research, projects, experience, and achievements. The site is designed for clarity, easy navigation, and accessibility.

---
## Features

- Responsive Design: Adapts to all screen sizes with a tabbed layout for quick navigation.
- Sections: Profile, Publications, Engineering & Code Projects, Experience, Blogs, Achievements, and Organizations & Membership.
- Portfolio Cards: Organized display of research, projects, and work experiences with concise descriptions and external links.
- Contact & CV Links: Direct links to Email, Google Scholar, GitHub, LinkedIn, ResearchGate, and a downloadable CV.
- Certifications & Memberships: Displayed with uniform logos for a clean, professional look.
- Accessibility & SEO: Semantic HTML, alt text for images, and meta tags for better search visibility.
